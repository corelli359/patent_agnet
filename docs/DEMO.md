# 🎯 演示：大模型+敏感词专利检索

## 快速启动

### 方式1：命令行演示
```bash
# 运行演示脚本（会自动检索并生成报告）
python scripts/demo_search.py
```

### 方式2：Web界面演示

#### 步骤1：启动API服务
```bash
python backend/run_server.py
```

#### 步骤2：打开前端界面
```bash
# Mac
open frontend/index.html

# 或直接在浏览器打开
# file:///Users/weipeng/Desktop/农行专利/2026/patent_agnet/frontend/index.html
```

#### 步骤3：在Web界面中
1. 输入关键词：`大模型 and 敏感词`
2. 设置起始日期：`2020-01-01`
3. 设置国家：`CN`
4. 点击"开始检索"
5. 查看结果和分析报告
6. 切换到"检索历史"tab查看历史记录

## 注意事项

### 性能
- 每次检索约30-60秒（Selenium加载时间）
- 30条结果建议分批检索（避免超时）
- 推荐先用10条测试，确认后再增大

### Google Patents限制
- 建议每次检索间隔3-5秒
- 避免短时间大量请求
- 使用代理可提高稳定性
